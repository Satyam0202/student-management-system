## How to Add Screenshots

### Step 1: Take Screenshots
1. Run your application:
   ```bash
   python student_management_system.py
   ```

2. Take screenshots of:
   - Main dashboard (all tabs visible)
   - Student management form (with sample data)
   - Records view (with student list)
   - Analytics dashboard (with statistics)

3. Use these keyboard shortcuts:
   - **Windows**: `Win + Shift + S` or `PrtScn`
   - **Mac**: `Cmd + Shift + 4`

### Step 2: Save Screenshots
Save your screenshots in this folder with these names:
- `dashboard.png` - Main application window
- `student-form.png` - Student management form
- `records-view.png` - Records and search view
- `analytics.png` - Analytics dashboard

### Step 3: Upload to GitHub
1. Go to your GitHub repository
2. Navigate to the `screenshots` folder
3. Click **Add file** ‚Üí **Upload files**
4. Drag and drop your screenshots
5. Commit changes

### Step 4: Verify
- Go to your repository's main page
- Scroll down to the Screenshots section in README
- Your images should now be visible!
